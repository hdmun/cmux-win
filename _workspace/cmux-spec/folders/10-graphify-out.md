# `graphify-out`

## 역할 / 목적

**생성물 전용 디렉터리**. `graphify` 에이전트 스킬이 코드베이스를 분석해 생성한 지식 그래프 캐시 파일들을 저장한다.

## 주요 내용

`graphify-out/cache/` 하위에 해시 이름의 JSON 파일 다수. 에이전트 실행 시 자동 생성·갱신된다.

## 편집 지침

**무시**. 코드 탐색·개발·테스트 작업 시 완전히 스킵한다. 삭제해도 다음 graphify 실행 시 재생성된다. 현재 `cmux/.gitignore`에는 별도 무시 규칙이 없으므로, 생성 후 작업트리에 남을 수 있다.
